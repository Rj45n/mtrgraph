# mtrgraph
All-in-one network &amp; S3 latency diagnostic toolkit: colored MTR + HTTP/TLS/TTFB probes + S3 SigV4 ops (AWS, MinIO, Scaleway, OVH, …) + concurrent bench. Real-time web dashboard with KPIs, per-IP analysis, baseline alerts &amp; Slack webhooks. Self-contained, SQLite/WAL, Docker, K8s-ready.
